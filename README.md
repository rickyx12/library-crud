#### INSTALLATION  
* `git clone https://github.com/rickyx12/library-crud.git projectname`  
* `cd projectname`  
* `composer install`  
* Create database and inform .env  
* `php artisan migrate --seed`  
* proceed to `http://localhost/projectname/public/book`
